# 📣 THÔNG TIN BUỔI 4

FPT Telecom gửi tới bạn thông tin chi tiết Buổi 4 - Embedded Bootcamp 2026 để bạn chủ động sắp xếp tham gia:

1. Chủ đề: Active Kernel (AK): Khái niệm, tính năng \& cách sử dụng

**Nội dung chính:**

* Cách tạo và quản lý Task, Các loại Message, Cách sử dụng Timer, State Machine.
* Cách sử dụng các công cụ hỗ Debug: Log \& Fatal, Command Line, Realtime Event \& Record Event.
