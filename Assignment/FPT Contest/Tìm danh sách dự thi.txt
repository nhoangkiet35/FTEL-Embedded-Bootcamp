Tìm danh sách dự thi
Bạn có danh sách dự thi của các thí sinh trong trường cho 4 môn: Toán, Văn, Anh, Vật lý. Mỗi môn được quản lý bằng một Linked List.

Mỗi Node trong Linked List có các thuộc tính:
- name: tên thí sinh (ví dụ: "NguyenVanA")
- subject: tên môn ("toan", "van", "anh", "vatly")
- className: lớp (ký tự 'A', 'B', 'C', 'D')
- next: con trỏ đến Node tiếp theo

Cho 4 Linked List tương ứng với 4 môn:
- mathList (Toán)
- literatureList (Văn)
- englishList (Anh)
- physicsList (Vật lý)

Hãy gộp (merge) 4 Linked List này thành một Linked List duy nhất.

Quy tắc:
1. Thứ tự gộp linked list phải là: mathList → literatureList → englishList → physicsList
2. Giữ nguyên thứ tự xuất hiện của thí sinh trong từng danh sách.
3. Tái sử dụng các Node đã có, không cần tạo Node mới.

Mô tả test case:
- Dòng đầu: Số lượng thí sinh của từng môn lần lượt là Toán-Văn-Anh-Vật Lý.
- Các dòng tiếp theo là danh sách thí sinh theo từng môn.
Kết quả:
- Danh sách thí sinh theo định dạng [Tên thí sinh]-[Môn thi]-[Lớp]

Ví dụ:
2 2 1 2
Bao A

Student* makeNode(const string& name, const string& subject, char className) {
    return new Student{name, subject, className, nullptr};
}

void appendNode(Student*& head, Student*& tail, Student* node) {
    if (!head) head = tail = node;
    else { tail->next = node; tail = node; }
}

void buildList(int count, const string& subject, Student*& head, Student*& tail) {
    for (int i = 0; i < count; i++) {
        string name;
        char className;
        if (!(cin >> name >> className)) return;
        appendNode(head, tail, makeNode(name, subject, className));
    }
}

void printList(Student* head) {
